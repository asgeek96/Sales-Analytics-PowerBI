# Sales Analytics Complete BI Project

## Overview

End-to-end Sales Analytics solution using SQL Server and Power BI.

## Project Structure

-   sales_data.csv → Raw dataset
-   sales_queries.sql → SQL table creation & analytics queries
-   PowerBI_Documentation.md → DAX & Power Query code

## Steps to Use

1.  Import sales_data.csv into SQL Server.
2.  Run sales_queries.sql to create and analyze data.
3.  Open Power BI Desktop.
4.  Load the CSV using provided Power Query code.
5.  Create DAX measures from documentation.
6.  Build dashboard pages as suggested.

## KPIs Included

-   Total Sales
-   Total Profit
-   Profit Margin
-   YTD Sales
-   Customer Ranking
